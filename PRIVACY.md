# Privacy

Mac Drag Scroll is a local macOS utility.

## What The App Needs

Mac Drag Scroll needs Accessibility permission to detect the middle mouse button globally and send scroll events to the active app.

## What The App Does Not Do

- It does not record keystrokes.
- It does not read document contents.
- It does not track browsing history.
- It does not sell or share personal data.

## Network Access

Mac Drag Scroll may contact GitHub when you check for updates or when automatic update checks are enabled. That request is used only to compare the installed version with the latest GitHub release.

## Local Settings

Settings such as speed, visualizer size, launch-at-login preference, and excluded apps are stored locally on your Mac.
